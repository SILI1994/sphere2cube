# sphere2cube

Tools to convert equirectangular projection 360 panoramas in 6 faces of the cube.

Migrate from a [python version](https://github.com/flash286/sphere2cube).

## Requirement

* opencv2
* cmake

## Compiler and Run

### Compile

```
mkdir build
cd build
cmake ..
make
```

### Run

```
./main [Panorama Filename]
```
